# Entrenadores y Premios - Mundial 2026

## 📋 Descripción
Aplicación de escritorio en **Visual Basic .NET** para gestionar información de entrenadores y premios del Mundial 2026.

## ✨ Características
- ✅ Formulario de Bienvenida
- ✅ Menú Principal interactivo
- ✅ Registro de Entrenadores
- ✅ Gestión de Premios
- ✅ Validaciones de datos
- ✅ Interfaz profesional

## 🚀 CÓMO ABRIR EL PROYECTO

### 1. Descarga los archivos
- Haz clic en **Code** → **Download ZIP**
- Descomprime la carpeta

### 2. Abre en Visual Studio
- Abre Visual Studio
- **File** → **Open** → **Project/Solution**
- **SELECCIONA: `EntrenadorPremios.sln`** (este es el archivo importante)

### 3. Ejecuta
- Presiona **F5** o el botón **Start**
- ¡La aplicación se abrirá correctamente!

## 📁 Archivos del Proyecto
- **EntrenadorPremios.sln** ← ABRE ESTE
- **EntrenadorPremios.vbproj** ← Configuración del proyecto
- **frmBienvenida.vb** ← Pantalla de inicio
- **frmMenuPrincipal.vb** ← Menú principal
- **frmEntrenadores.vb** ← Gestión de entrenadores
- **frmPremios.vb** ← Gestión de premios
- **Module1.vb** ← Punto de inicio

## 🎯 Funcionalidades

### Sección Entrenadores
- Agregar nuevo entrenador
- Ingresar: Nombre, País, Equipo, Años de experiencia
- Ver lista de entrenadores
- Limpiar formulario

### Sección Premios
- Agregar nuevo premio
- Ingresar: Tipo, Equipo ganador, Monto, Descripción
- Ver lista de premios
- Limpiar formulario

## 📝 Conceptos VB.NET Aplicados
- Condiciones (If/Else)
- Controles dinámicos (TextBox, ComboBox, NumericUpDown, ListBox, Button, Label, Panel)
- Eventos (Click, Load)
- Bucles (For/Each)
- Listas (List(Of String))
- Validaciones de datos
- Navegación entre formularios
