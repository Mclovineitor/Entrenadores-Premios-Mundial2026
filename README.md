# Entrenadores y Premios - Mundial 2026

## 📋 Descripción
Aplicación de escritorio en **Visual Basic .NET** para gestionar información de entrenadores y premios del Mundial 2026.

## ✨ Características
- ✅ Formulario de Bienvenida animado
- ✅ Menú Principal interactivo
- ✅ Registro de Entrenadores
- ✅ Gestión de Premios
- ✅ Validaciones de datos
- ✅ Interfaz amigable y profesional

## 🎯 Funcionalidades Implementadas

### Programación VB.NET
- Condiciones (If/Else)
- Controles variados (TextBox, ComboBox, NumericUpDown, ListBox, Button, Label, Panel)
- Eventos (Click, Load)
- Bucles para cargar datos
- Contenedores (Panel)
- Variables y Listas

### Módulos
1. **frmBienvenida.vb** - Pantalla de inicio
2. **frmMenuPrincipal.vb** - Menú de opciones
3. **frmEntrenadores.vb** - Gestión de entrenadores
4. **frmPremios.vb** - Gestión de premios

## 🚀 Instalación

### Requisitos
- Visual Studio 2019 o superior
- .NET Framework 4.7.2 o superior

### Pasos
1. **Descarga el repositorio**
   - Click en **Code** → **Download ZIP**
   - Descomprime la carpeta

2. **Abre en Visual Studio**
   - Abre Visual Studio
   - Archivo → Abrir → Proyecto/Solución
   - Selecciona el archivo **EntrenadorPremios.sln**

3. **Ejecuta el proyecto**
   - Presiona **F5** o **Start**
   - ¡Listo! La aplicación se abrirá

## 👨‍💻 Uso

### Acceso a Entrenadores
1. Ingresa al sistema desde la pantalla de bienvenida
2. Selecciona "Gestionar Entrenadores"
3. Completa los datos: Nombre, País, Equipo, Años de Experiencia
4. Haz clic en "Agregar"
5. Los entrenadores se mostrarán en la lista

### Acceso a Premios
1. Desde el menú principal, selecciona "Gestionar Premios"
2. Selecciona el tipo de premio (Campeón, Subcampeón, etc.)
3. Ingresa el equipo ganador y el monto
4. Añade una descripción
5. Haz clic en "Agregar"
6. Los premios se mostrarán en la lista
